However, there was a problem - electronic mail uses html, and I did not know how to make one from scratch. Seeing this, I had to learn how to make this electronic mail from scratch by watching multiple youtube videos.

This is my very first attempt at this, so I hope those who would like to learn how to do this for the first time can benefit a lot from this version.
